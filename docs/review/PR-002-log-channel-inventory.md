## Пригодность каналов

- Для `ModelRate`: найдены `RATE`, `RCOU` и `ATT` во всех трех журналах.
- Для `Model6DOF`: найдены `ATT`, `RATE`, `RCOU`, `GPS`, `POS`, `XKF1`...`XKF4`, `IMU`, `BARO`.
- Для `BatteryModel`: найдено `BAT`; фактические записи `ESC` отсутствуют.
- Для оценки адекватности по ВБ: найдены `MODE`, `ATT`, `RATE`, `RCOU`, `BAT`, `GPS`, `POS`, `XKF1`...`XKF4`.

## Вывод

Фактический состав каналов достаточен для подготовки данных модели угловых скоростей и предварительной подготовки данных `Model6DOF`. Для модели аккумулятора доступен `BAT`, но нет фактических записей `ESC`. Следующий PR должен выделить участки ВБ, подтвердить высоту и режимы по журналам, затем выполнить идентификацию параметров на выбранных участках.
