# Методика оценки адекватности по бортовому журналу

## Состав валидационного базиса первого уровня

Валидационный базис первого уровня включает табличные бортовые журналы ArduPilot, предварительно приведенные пользователем к CSV, MAT, table MATLAB или timetable MATLAB.

Файл .BIN на первом этапе должен быть предварительно приведен к табличному виду штатными средствами пользователя. Прямое чтение .BIN оставлено как адаптер-заготовка.

Данная проверка выполнена по данным бортового журнала и не является полной независимой валидацией реального изделия по внешним средствам измерений.

## Разделение данных

Данные для идентификации параметров и данные для проверки должны быть разными. Если используется один набор данных, он делится по времени. По умолчанию применяются короткие участки от 2 до 10 секунд.

На участках проверки параметры компьютерной модели не настраиваются.

## Показатели расхождения

Рассчитываются:

- среднеквадратическое отклонение;
- среднее абсолютное отклонение;
- максимальное абсолютное отклонение;
- нормированное среднеквадратическое отклонение;
- показатель соответствия.

Показатель соответствия:

```text
100 * (1 - norm(y_ref - y_calc) / norm(y_ref - mean(y_ref)))
```

## Матрица валидации

Минимальная матрица валидации содержит столбцы:

- идентификатор проверки;
- режим;
- источник данных;
- участок времени;
- проверяемые параметры;
- показатели расхождения;
- критерий приемлемости;
- результат;
- вывод;
- ограничения.

## Предварительные критерии приемлемости

По умолчанию заданы ориентировочные критерии:

- угловые скорости крена и тангажа: показатель соответствия не менее 70%;
- угловая скорость рыскания: показатель соответствия не менее 60%;
- углы крена и тангажа: среднеквадратическое отклонение не более 3 градусов;
- угол курса: среднеквадратическое отклонение не более 5 градусов.

Эти критерии являются предварительными и подлежат уточнению после накопления валидационного базиса.

## Структура отчета

Отчет оценки адекватности включает:

- наименование и идентификатор изделия;
- версию модели;
- сведения о среде MATLAB;
- сведения о конфигурации изделия;
- хэш конфигурации;
- хэш исходного набора данных;
- хэш обработанного набора данных;
- перечень использованных каналов;
- предупреждения по качеству данных;
- параметры модели;
- участки идентификации;
- участки проверки;
- таблицу показателей расхождения;
- графики сопоставления расчетных и зарегистрированных величин;
- матрицу валидации;
- ограничения применимости;
- заключение.
