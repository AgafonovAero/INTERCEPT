# Договор на данные бортового журнала

## Форматы первого этапа

Поддерживаются CSV, MAT, table MATLAB и timetable MATLAB. Файл .BIN на первом этапе должен быть предварительно приведен к табличному виду штатными средствами пользователя.

## Время

Ожидается один из столбцов времени:

- `TimeUS`, микросекунды;
- `TimeMS`, миллисекунды;
- `TimeS`, секунды;
- `Time`, секунды или время MATLAB;
- `timestamp`;
- `t`;
- `time`.

Препроцессор приводит время к секундам, сортирует записи, удаляет дубли времени, выявляет пропуски и приводит записи к равномерной сетке. По умолчанию используется 100 Гц.

## Поля бортового журнала

Желательные каналы:

- `ATT.Roll`, `ATT.Pitch`, `ATT.Yaw`;
- `ATT.DesRoll`, `ATT.DesPitch`, `ATT.DesYaw`;
- `RATE.R`, `RATE.P`, `RATE.Y`;
- `RATE.RDes`, `RATE.PDes`, `RATE.YDes`;
- `RATE.ROut`, `RATE.POut`, `RATE.YOut`, `RATE.AOut`;
- `RCOU.C1`, `RCOU.C2`, `RCOU.C3`, `RCOU.C4`;
- `BAT.Volt`, `BAT.Curr`, `BAT.CurrTot`, `BAT.EnrgTot`;
- `ESC.RPM`, `ESC.Curr`, `ESC.Volt`.

Отсутствующие необязательные каналы отражаются в отчете качества данных. Выполнение не должно аварийно завершаться только из-за отсутствия необязательных каналов.

## Единицы измерения

- Углы `ATT.*` ожидаются в градусах и приводятся к радианам для расчета.
- Угловые скорости `RATE.*` ожидаются в градусах в секунду и приводятся к радианам в секунду для расчета.
- `RCOU.C1-C4` ожидаются в микросекундах и нормируются в диапазон от 0 до 1.
- Параметры нормирования задаются в конфигурации изделия.

## Правила именования столбцов

Имена столбцов должны сохранять имена полей бортового журнала ArduPilot, например `RATE.R` и `RCOU.C1`. При чтении CSV используется сохранение исходных имен столбцов.

## Требования к качеству данных

Данные должны:

- иметь возрастающее время;
- не содержать дубли времени после подготовки;
- иметь достаточную длительность для разделения на участки идентификации и проверки;
- содержать каналы угловых скоростей для идентификации модели угловых скоростей;
- содержать `RATE.ROut/POut/YOut` либо `RCOU.C1-C4` для входных воздействий.
