# Документ валидационного базиса

Пользователь должен положить документ валидационного базиса сюда:

`docs/validation_basis/ValidationBasis_v0.pdf`

PDF валидационного базиса используется как исходный организационно-методический документ.

Формализованный реестр случаев хранится в `config/validation_basis_v0.json`.

При изменении PDF необходимо обновлять JSON-реестр и фиксировать изменение в журнале конфигурации.

Если документ содержит чувствительную информацию, храните его в закрытом репозитории или не добавляйте в Git.
