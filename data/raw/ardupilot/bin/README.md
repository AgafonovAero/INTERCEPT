# Бортовые журналы ArduPilot DataFlash

Пользователь должен положить реальные бортовые журналы ArduPilot DataFlash сюда:

- `data/raw/ardupilot/bin/00000006.BIN`
- `data/raw/ardupilot/bin/00000007.BIN`
- `data/raw/ardupilot/bin/Alt_50m.bin`

Файлы BIN на текущем этапе используются как исходные данные для будущего анализа структуры бортовых журналов. В текущем PR полноценное чтение BIN не реализуется.

Следующий этап проекта — анализ реальной структуры этих BIN журналов и реализация адаптера чтения DataFlash в MATLAB-последовательность обработки данных.

Если репозиторий публичный, сырые журналы не рекомендуется добавлять в обычный Git, потому что они могут содержать координаты, время полета, параметры изделия и другую чувствительную информацию.

Для хранения журналов в репозитории используйте Git LFS или закрытый репозиторий. Если журналы не добавляются в Git, достаточно хранить их локально по указанному пути.
