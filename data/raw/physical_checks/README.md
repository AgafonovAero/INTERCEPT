# Физическая проверка конфигурации изделия

В этот каталог помещаются результаты физической проверки объекта моделирования `qc_ardupilot_v0`.

Нужны следующие проверки:

- соответствие номера мотора физическому положению;
- соответствие канала `RCOU` или `SERVO` физическому мотору;
- направление вращения каждого мотора;
- тип установленного винта;
- направление установки винта;
- соответствие носа изделия связанной системе координат.

Проверка выполняется только уполномоченным специалистом организации. Проверку направления вращения следует выполнять безопасным способом согласно внутренним процедурам организации. При проверке направления вращения без необходимости тяги воздушные винты должны быть сняты.

Результаты должны быть сохранены в структурированном виде. Шаблоны:

- `physical_configuration_check_template.json`;
- `motor_spin_check_template.csv`;
- `motor_order_check_template.csv`;
- `propeller_installation_check_template.csv`.

Реальные файлы проверки могут содержать чувствительные сведения и не добавляются в обычный Git.
