# Исходные данные

В этот каталог помещаются исходные наборы данных для препроцессора. На первом этапе рабочими форматами являются CSV, MAT, table MATLAB и timetable MATLAB.

Сырые бортовые журналы ArduPilot DataFlash BIN размещаются в `data/raw/ardupilot/bin/`. В публичный Git такие файлы по умолчанию не добавляются.
