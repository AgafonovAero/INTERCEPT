# Стендовые данные винтомоторной группы

В этот каталог помещаются стендовые данные винтомоторной группы объекта моделирования.

Для первичной модели нужны:

- команда `PWM`, `RCOU` или `normalized_input`;
- напряжение;
- ток;
- тяга;
- обороты, если есть;
- температура, если есть;
- тип двигателя;
- тип винта;
- тип ESC;
- тип аккумулятора или источника питания.

Шаблоны:

- `propulsion_static_test_template.csv`;
- `battery_discharge_test_template.csv`;
- `esc_current_voltage_template.csv`.

Реальные стендовые данные могут содержать чувствительную информацию и не добавляются в публичный Git.
